        [뷰포트(viewport)]
		1. 웹 페이지에서 사용자가 화면상에 볼 수 있는 영역
		2. 여러 디바이스(스마트폰, 테블릿, 컴퓨터 등등) 표시 영역 다르다.
		3. 여러 장치의 크기에 맞게 웹 페이지의 크기를 제어해서 화면에 보여 줄 필요가 있다.
		4. 뷰포트 설정
		
		
		width=device-width 너비는 장치의 너비 맞게
		initial-scale=1.0 비율 1.0 조정(축소, 확대)
    
        [미디어 쿼리]
        - css3 도입된 css 기술
        - 여러 장치 마다 서로 다른 스타일을 적용할 필요 o
        - 반응형 웹
        - @media 규칙
            미디어 쿼리 형식
            @media 조건식(참 / 거짓) {
				css 코딩;
			}
			
			@media [only|not 미디어타입 and ] (조건식 true.false) {
				css 코딩;
			}
            미디어타입 : [all](기본값) screen, print 등등
			
		- 2) link 태그

            [css flex]
            유연성 상자 레이아웃 모듈
            float, position을 사용하지 않고도 유연한 반응형 레이아웃 구조를 쉽게디자인할 수 있다

            1. 플렉스 항목(item) 선언
                - order
                - flex-grow
                - flex-shrink
                    움츠러든다
                    축소된다
                - flex-basis
                - flex 약식
                    flex: flext-grow flex-shrink flex-basis
                - align-self

            2. 플렉스 컨테이너 선언
                - flex-direction
                - flex-wrap
                - flex-flow
                - justify-content
                - align-items
                - align-content
