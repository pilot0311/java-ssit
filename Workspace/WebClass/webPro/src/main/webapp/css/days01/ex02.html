[jquery,CSS]의 선택자(selector)
    1) 태그명, #아이디, .클래스명

    [css 특이성]
    - 동일한 요소를 가리키는 두 개 이상의 CSS규칙이 있는 경우
        가장 높은 특이성을 가진 선택자에 의해 css적용
    - 특이성 == 스타일 적용 점수/순위
    (암기)
        태그명              1점
        클래스명            10점
        아이디명            100점
        태그명.아이디명     101점
        태그명.클래스명     11점
        인라인식 css        1000점
        예)
            p.box.first :    1+10+10=21점
            #nav div.#demo : 100+1+100=201점
        
        *                   0점

    - (기억) 한 가지 예외 있다
    !important: 속성뒤에 붙일경우 무조건 적용

    -   1) 인라인 CSS적용
        2) 내부/외부 CSS적용 방식(헤드 코딩 순서)
