[background-origin속성]
    - 배경 이미지의 위치를 지정하는 속성
    - 속성값
        1) border-box           : 테두리 왼쪽 상단
        2) padding-box (기본값)  : 패딩 왼쪽 상단 
        3) content-box          : 콘텐츠 왼쪽 상단
    - 배경 이미지의 위치 background-position 속성

    [background-clip 속성]
    - 배경의 페이팅 영역을 지정하는 속성
        border-box(기본값)  : 배경이 테두리의 바깥쪽 가장자리에 그려진다
        padding-box         : 배경이 패딩의 ''
        content-box         : 콘텍츠 박스 내에서 배경이 그려진다
