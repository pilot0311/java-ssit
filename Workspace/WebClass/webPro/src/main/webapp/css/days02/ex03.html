    1. 패딩: 테두리 + 간격 + 내용(content)
    2. 
        padding-top
        padding-right
        padding-bottom
        padding-left

        padding : 약식
    3. 단위: px,pt,cm,%,inherit (음수X)
    4. (주의)
        패딩은 요소의 너비에 영향을 준다
