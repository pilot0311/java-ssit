[display 속성]
    1. 레이아웃(배치)을 제어하기 위한
    가장 중요한 css 속성이다
    2. 기본값: block, inline
    block - div,p,header,hr,hn 등등
    특징)   항상 새로운 라인에서 시작
            부모 너비의 100%

    inline - span, a, img 등등
    특징)   새로운 라인에서 시작X
            너비는 필요한 내용(content)
            만큼만 차지
    3. 요소가 표시되는 여부를 지정하는 속성
    4. display: none; : 숨기기, 요소가 삭제된 것은 아님
