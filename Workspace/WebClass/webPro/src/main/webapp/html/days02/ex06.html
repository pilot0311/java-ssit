    - html/images 폴더 생성
    - 웹 페이지에 이미지 추가
    - 웹 페이지에 이미지 삽입x
        단지 이미지가 웹페이지에 연결O
    - 2가지 필수 속성 지정
        src - 이미지 경로
        alt - 이미지 대체할 텍스트
    - width(너비), height(높이)를 지정하지 않으면
        원래 이미지의 너비, 높이로 설정

    - 항상 이미지의 너비, 높이 속성을 지정하자
        이유: 너비,높이가 지정되지 않으면
        이미지가 로딩되는 동안 웹 페이지가 깜빡임이
        발생하기 때문에...
