  1. 인터넷(Internet)
    • 전 세계를 연결하는 국제 정보 통신망
    • ARPA(Advanced research projects Agency)가 1969년에 웹의 모태가 되는 ARPANET 개발.
    • 군사용과 민간용 구분
    • 민간용으로 사용되던 네트워크가 현재의 인터넷이 되었다
  2. 웹(web)
    1. 인터넷에서 작동되는 서비스
    2. 인터넷에 연결된 컴퓨터를 통해서 사람들이 정보(영상,문자 등등)을 공유하는 공간
    3. 1991년 등장
    4. 1989년 유럽 공동 원자핵 연구소(CERN)의 "팀버너스 리"기 하이퍼 링크를 포함하는 문서의 개념을 제안 -> 1991년 월드 와이드 웹을 개발해서 배포
  3. W3C(World Wide Web Consortium)
    1. 국제 웹 표준화 단체(기구)
    2. 1994년 10월 설립(팁버너스리 창설)
    3. html5,css3 등등 표준화 단체
    4. WWW의 표준안 제작과 새로운 표준안 제안 기술의 공유를 통해 WWW의 기술력,사회적확장을 위해 구성된 세계적 단체
  4. 웹 브라우저
    1. 인터넷에서 정보를 검색(공유)하는 응용 프로그램
    2. 최초의 웹 브라우저 - 모자이크(1993) -> 후에 "넷 스케이프" 발표
    3. W3C의 표준을 무시하고 경쟁 MS사 - 인터넷 익스플로러(IE) 넷스케이프사 - 넷스케이프 : :
    4. 1차 웹브라우저 전쟁 - 웹 브라우저 사이에 발생하는 기술적인 전챙 (1994~1996)->MS사의 IE 승리
  5. 플러그인
    • 웹 브라우저와 연동되는 특정 프로그램을 사용자의 pc에 추가로 설치하여 웹 브라우저의 기능을 확장하는 방법
    • 웹의 발전 속도를 웹브라우저가 따라가지 못해서 기업 개발
    • MS사의 엑티브엑스, 어도비사의 플래시 등등
  6. 웹 1.0 웹 2.0(사용자 새로운 컨텐츠를 창조) 웹 3.0(인터넷 엄청난 양의 정보 -> 필요한 정보/지식을 추출해서 보여 주는 맞춤형 시대)
  7. 플러그인 문제: 액티브X 설치하면 웹 사이트가 무거워짐 HTML 표준 제안서 제출 -> 거절 MS사(IE)
  8. MS사(IE) 제외한 그 외의 웹 브라우저 재공 업체 - WHATWG
  9. WHATWG(Web Hypertext Application Technology Working Group)
  10. 2004.6 html5 표준을 제정
  11. 새로운 웹 표준인 Web Application 1.0 표준을 작성
  12. W3C - Web Application 1.0 표준을 채택
  13. 웹 표준 변화 -W3C가 Web Application 1.0 표준 -> html5 표준으로 변경 -W3C + WHATWG 함께 html W/G 결성 -> html5 웹 표준 작성 - html5 표준은 현재도 작성중
  14. 2차 웹 브라우저 전쟁이 시작 -IE만 최신 표준을 지원하지 못하는 상황 발생 -웹 브라우저의 점유율 뒤집히는 상황 발생
  15. html 역사
